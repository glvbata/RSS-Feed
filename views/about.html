About BFAM RSS

This application allows the user to add any RSS feeds they want via the RSS source feed link. After initial registration, you will have the default RSS feeds which are gaming feeds until you add your own custom feeds.


Also, guests (non users) will have the gaming feeds by default as well.


Here are some samples:
  • http://www.cnn.com/services/rss/
  • http://www.gamespot.com/feeds/
  • http://www.fashionnstyle.com/rss/
  • http://espn.go.com/espn/news/story?page=rssinfo
